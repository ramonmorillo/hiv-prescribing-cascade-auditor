HIV Prescribing Cascade Auditor v1.0 — Clinical decision-support tool for detecting potential prescribing cascades in people living with HIV (PLHIV). / Herramienta de apoyo a la decisión clínica para detectar posibles cascadas de prescripción en personas que viven con el VIH.

Author / Autor: Ramón Morillo  ·  Architecture / Arquitectura: Local-first, no external API  ·  Reference / Referencia: Conceptually based on Rochon & Gurwitz (BMJ, 1997).
